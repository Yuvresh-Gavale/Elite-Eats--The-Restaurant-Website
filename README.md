# 🍽️ Elite Eats Restaurant Website

Elite Eats is a modern and responsive restaurant website designed to provide an engaging dining experience online. The website features a stunning hero section, signature menu, restaurant story, contact information, and a reservation form. Built entirely with HTML and CSS, this project focuses on clean design, responsiveness, and user-friendly navigation.

✨ Features
- Responsive Navigation Bar
- Attractive Hero Section
- Signature Menu Showcase
- About Us Section
- Reservation Form
- Contact Information
- Interactive Hover Effects
- Mobile-Friendly Design

🛠️ Technologies Used
- HTML5
- CSS3
